# Receipt-Tracker

## App Descriptiopn
- A webapp that lets users scan receipts to automatically, categorize purchases, and track spending. 

Each receipt is analyzed and cateforizerd into groups. The app provides a clear breakdown of spending per category. The app provides users ease of organizing by sorting and filtering expenses. The platform features responsive designs whether it is accessed through a browser or mobile app.

## Features

# Receipt Scanning
- Take a photo of a receipt using your device
- Automatically extracts item names and prices using OCR (Optical Character Recognition)

# Smart Categorization
- Classifies purchases into categories

# Expense Tracking
- Displays total amount per receipt
- Aggregates totals by category
- Provides overall spending summaries

# Sort and Filter
- Sort expenses by individual categories
- Combine categories to analyze grouped spending
- Quickly view totals for selected filters

## Platform

# Web application
- Accessible via modern browsers
- Responsive UI for desktop and mobile

# Mobile Application
- Native or cross-platform support for on-the-go receipt scanning

## How it works

- User uploads or captures a receipt image
- The system processes the image using OCR
- Extracted text is parsed into structured data
- Items are categorized automatically
- Totals are calculated and stored
- Data is displayed in dashboards with filtering and sorting options
